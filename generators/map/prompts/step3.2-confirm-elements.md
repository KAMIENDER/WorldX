## 你的任务

快速确认标注是否大致正确。**你不需要给出修正坐标**，只需要判断哪些元素有明显问题。

## 判断标准

### 通过（可接受）
- 标注框大致覆盖了该元素的主体
- 偏小一些完全可以接受（元素标注宁小勿大）
- 边缘轻微偏移均可接受
- 不要求像素级精确

### 不通过（明显有问题）
- 标注框**完全偏离**了目标元素（标在了错误的位置）
- 标注框覆盖了大面积的非相关区域
- 元素 id 与实际位置明显不符

### 容忍范围
- 框比理想范围偏小 → 通过
- 框稍微压到旁边的道路边缘 → 通过

## 返回格式

请返回 JSON（不要包含其他内容）：

如果所有元素均可接受：
```json
{ "pass": true, "problematic_element_ids": [] }
```

如果有明显问题的元素：
```json
{ "pass": false, "problematic_element_ids": ["有问题的元素id1", "有问题的元素id2"] }
```

## 重要
- 只标记**明显错误**的元素，不要标记"不够完美但基本正确"的元素
- 被标记的元素会被系统直接移除，所以请谨慎判断
- 如果所有元素都大致正确，请返回 pass: true
